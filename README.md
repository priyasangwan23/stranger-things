# 🎬 Stranger Things – Hero Animation Website

A modern **React-based website** inspired by *Stranger Things*, featuring smooth animations using **Framer Motion** and built with **Vite**.

---

## 🚀 Live Demo

👉 https://priya-stranger-things.netlify.app/

---

## 🛠 Tech Stack

- ⚛ **React 18**
- ⚡ **Vite**
- 🎞 **Framer Motion**
- 🎨 **CSS**
- 🌐 **Netlify** (Deployment)

---

## ✨ Features

- Smooth and engaging animations  
- Interactive hero section  
- Modern UI design  
- Fully responsive layout  
- Fast performance with Vite
